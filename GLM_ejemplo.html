GLM ejemplo de uso de Tlamatini

Modelos lineales generalizados (GLM)

Este paquete contiene funciones tiles para el anlisis de datos, principalmente para modelos lineales (LM, GLM, GLMM). Aunque est pensado para usarse en Ciencias Biolgicas, su aplicacin se extiende a cualquier rea del conocimiento que requiera anlisis estadsticos. Es una compilacin de funciones tiles creadas por mi o por alguien ms, que facilitan el anlisis de datos: exploracin y transformaciones de datos, revisin de los supuestos de los modelos lineales (LM, LMM, GLM, GLMM), exportacin de tablas, entre otros.

Si quieres ms informacin sobre el paquete y las actualizaciones, visita la pgina: https://mariosandovalmx.github.io/tlamatini-website/

Para citar este paquete: Mario A. Sandoval-Molina (2021). tlamatini: Funciones utiles para biologxs y ecologxs confundidos con los modelos lineales. R package version 0.1.

Si quieres conocer ms de mi trabajo, visita mi pagina web: https://mariosandovalmx.github.io/ecology/

Podemos ajustar un modelo lineal generalizado (GLM). Con esta base de datos vamos a ajustar dos modelos lineales generalizados, el primero con la distribucin gaussiana y logaritmo como funcin de liga. El segundo con distribucin gaussiana y funcin de liga identity. Usaremos la longitud hocico-cloaca (LHC) como variable de respuesta. Adems, usaremos la variable Velocidad y Localidad como variables explicativas.


modelo  <- glm(LHC ~ Velocidad  + Localidad, family = gaussian("log"), data= df)
modelo2 <- glm(LHC ~ Velocidad  + Localidad, family = gaussian("identity"), data= df)
AIC(modelo, modelo2)
#>         df  AIC
#> modelo   6 77.1
#> modelo2  6 75.8
